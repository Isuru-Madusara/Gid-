## 📌 About

技道 Gidō is a free, browser-based training platform built for
international students and engineers who want to start a career
in Japan's ICT industry.

Everything runs in a **single HTML file** — no installation,
no frameworks, no backend required. Just open and go.

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/yourusername/gido-ict.git

# Open in browser — that's it
open gido_ict.html
```

No npm. No build step. No server needed.
